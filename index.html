JavaScript ES6 Features

Closure

A closure is a function that remembers the variables from its lexical scope even when the function is executed outside that scope.

 function createCounter() {
let count = 0;
return function () {
count++;
return \`Count: ${count}\`;
};
}          
const counter = createCounter();
console.log(counter());
console.log(counter());

Destructuring

Destructuring allows you to unpack values from arrays or properties from objects into distinct variables.

//Array destructuring
const [firstName, lastName] = ["John", "Doe"];
console.log(\`First Name: ${firstName}, Last Name: ${lastName}\`);               
//Object destructuring
const user = {username: "johndoe", email: "john@example.com"};
const {username, email} = user;
console.log(\`Username: ${username}, Email: ${email}\`);

Default Parameters

Default parameters allow you to set default values for function parameters.

function greet(name = "Guest") {
return \`Welcome, ${name}!\`;
}
console.log(greet());
console.log(greet("Alice"));

Rest Parameter

The rest parameter syntax allows you to represent an indefinite number of arguments as an array.

function calculateTotal(...prices) {
return prices.reduce((sum, price) => sum + price, 0);
} 
console.log(\`Total: $${calculateTotal(10.99, 20.99, 5.49)}\`);

Spread Operator

The spread operator allows you to expand elements of an array or object.

// Array spread
const fruits = ["apple", "banana", "cherry"];
const moreFruits = ["mango", ...fruits, "pineapple"];
console.log(moreFruits);              
// Object spread
const baseProduct = {id: 1, name: "Laptop"};
const updatedProduct = {...baseProduct, price: 1200};
console.log(updatedProduct);

Enhanced Object Literals

Enhanced object literals provide syntactic sugar for defining objects.

const title = "Developer";
const details = {
title,
introduce() {
return \`Hello, I am a ${this.title}.\`;
}
}; 
console.log(details.introduce());

Property Shorthand

Property shorthand lets you create object properties using variables with the same name.

const width = 100;
const height = 200;
const dimensions = {width, height};
console.log(dimensions);

Method Shorthand

Method shorthand allows you to define functions in objects without the function keyword.

const mathUtils = {
multiply(a, b) {
return a * b;
},
divide(a, b) {
return a / b;
}
};
console.log(mathUtils.multiply(4, 5));

Computed Property Names

Computed property names allow you to define object properties using expressions.

const key = "dynamicKey";
const dynamicObject = {
[key]: "Dynamic Value"
};              
console.log(dynamicObject.dynamicKey);
